# JSON-RPC Provider

Provider based on JSON-RPC HTTP requests.

### new JSONRPCProvider

Creates a new instance of the JSON-RPC Provider

#### Parameters

* `baseURL` **string** the JSON-RPC URL of the node

#### Usage

```ts
new JSONRPCProvider('http://staging.gno.land:36657');
// provider is created
```
