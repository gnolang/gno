# DAOkit Demo Templates

Collection of DAO examples made with [`/p/samcrew/daokit/`](/p/samcrew/daokit) that can be used as templates for building your own DAOs.

## Available Templates

- 1. **Simple DAO** [`/r/samcrew/daodemo/simple_dao/`](/r/samcrew/daodemo/simple_dao/): Basic DAO with roles and member voting.
- 2. **Custom Resource** [`/r/samcrew/daodemo/custom_resource/`](/r/samcrew/daodemo/custom_resource/): DAO with custom actions (blog posts example).
- 3. **Custom Condition** [`/r/samcrew/daodemo/custom_condition/`](/r/samcrew/daodemo/custom_condition/): DAO with custom voting rules (only roleless members vote).

## Sources 

- [`/p/samcrew/daokit`](/p/samcrew/daokit) - Core DAO framework and proposal system
- [`/p/samcrew/basedao`](/p/samcrew/basedao) - Member and role management system
- [`/p/samcrew/daocond`](/p/samcrew/daocond) - Voting conditions and governance rules
